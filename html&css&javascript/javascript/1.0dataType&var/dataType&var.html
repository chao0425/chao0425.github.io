js中的变量声明使用的是var(弱类型语言)
js中是不需要使用";"分号的
js中我们想要打印可以使用：console.log()，值得注意的是，打印结果我们要f12在浏览器的控制台才能看到

js中常见的类型有：数值类型、字符串类型、布尔类型、引用类型（Object）、函数类型（function类型）
逆天的js如果var声明变量，直接写一下举例吧
var string = "string"
string = "we can change"
string = 666
 这样的话一点毛病没有，从一开始的字符串类型变为了数值类型...
甚至一个变量可以重复声明，即前面
var name = "666"
接下来直接：
var name = 666 也没毛病
 对于我们想知道某个变量什么类型，可以用typeof
刚刚发现可以console.log(typeof "String") 也可以console.log(typeof("String")) 如果声明没有赋值，就是undefined，输出、类型都是undefined
如果赋值null了，值就是null，类型是object了。
至于命名风格等，可以参照Java。
