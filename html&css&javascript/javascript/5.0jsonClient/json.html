JSON格式仅仅是一个文本。是一种字符串语法格式。
在前后端交互的过程中，可以使用json格式来交互。

json格式可以如下
var str = '{"属性名":属性值, "属性名":属性值, ..., "属性名":{}, "属性名":[]}'
var personStr = '{"name":"Bob", "age":23, "hobby":["eat","sleep","game"] }'
 注意属性名必须要双引号包裹上，对于属性值，字符串就双引号，数字就不用处理。
var person = JSON.parse(personStr);
我们可以通过JSON.parse()将json格式的字符串转化为js的对象。
JSON.stringify(person)
stringify是可以将对象转化为json格式的
