事件

我们发现事件绑定的属性普遍属性名是on开头

事件是用户、浏览器的行为，发生一些行为时候，我们可以让js运行，称为事件的发生。当事件发生与js结合，称为事件的绑定。
对于事件，我们主要分为三类：1.鼠标点击，2.键盘输入，3.表单事件


使用了onclick绑定了js的hello()方法（双击）

使用了ondblclick绑定了js的方法（双击）

对于事件的绑定：
1.可以通过html属性绑定
一个属性可以绑定多个函数如：onclick="hello(),other()"
alert的弹窗是干扰我们点击效果的(单击双击同时有弹窗，那么双击永远都不会执行)
2.通过dom动态绑定
