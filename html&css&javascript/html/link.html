对于引入链接的基本操作就是a标签，然后属性名为href值为目标的URL
默认情况下，链接将以以下形式出现在浏览器中： 一个未访问过的链接显示为蓝色字体并带有下划线。 访问过的链接显示为紫色并带有下划线。 点击链接时，链接显示为红色并带有下划线。

对于a标签的属性，除了我们之前用过的href基础属性外，我们还可以定义链接打开的方式，如是新界面打开还是当前界面打开
使用target：定义链接的打开方式。
_blank: 在新窗口或新标签页中打开链接。target的用法，跳转链接再探
_self: 在当前窗口或标签页中打开链接（默认）。
_parent: 在父框架中打开链接。target的用法，跳转链接再探
_top: 在整个窗口中打开链接，取消任何框架。target的用法，跳转链接再探
