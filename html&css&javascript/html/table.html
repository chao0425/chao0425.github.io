表格是由table标签来定义的
table标签里面内嵌thead以及tbody，就是head头部以及body内容主体部分。
无论是在head还是body我们都是需要用tr来定义行，表示一行。是表示一行，一行里面有多少列是tr内部里面说的。
在tr里面我们继续th以及td。th是头部的，head嘛。对于td是data的。
第1列的th 第2列的th 第3行的th
data的第1列 data的第2列 data的第3列

如果是希望有边框，可以额外说：border="1"，至于里面的数字，数字越大，边框越宽。对于边框的颜色，在css里面设置border-color就可以了。border-collapse: collapse;可以将边框线设置为单实线，比较舒服。
