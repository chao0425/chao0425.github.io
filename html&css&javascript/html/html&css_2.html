HTML中，使用link引入css文件

暂时来看，使用link从外部引入css文件或者在head里面写css没什么大区别，因为我们只有一个css但是当css文件多了的时候，会显得很乱。即使只有一个css文件，那么写多了也很凌乱。
基本就这样了，更多的css应该在css部分学习. end

接下来是一些引入外部css的示例
在head中写：<link rel="stylesheet" href="css.css">
然后就是css的内容了
body {
background-color: #fffaf0;
font-size: 24px;
color: gray;
}

h1 {
text-align: center;
color: black;
}

h6 {
text-align: center;
}

b {
color: red;
}
