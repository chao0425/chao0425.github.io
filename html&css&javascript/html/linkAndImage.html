有些时候，我们不得不引入一些链接或者图片，无论是外部的还是本项目内的。

如果我们引入链接可以使用a标签，格式如：<a href="https://chao0425.github.io/html&css&javascript/">对外展示的名字</a>
这个是我github的一个链接
当然，我们也可以引入一些图片，格式如：<img src="项目的相对路径或者绝对路径" alt="当图像无法显示时显示这段文本" width="宽度" height="高度"/（这个斜杠没有也没啥毛病）>

对于video的话，则是video标签，然后属性填写长宽。以及controls按钮。具体的内容则是source标签，填写src源文件以及type文件类型

我不明白source标签的title有什么用，还有就是我发现视频没有找到alt的功能
