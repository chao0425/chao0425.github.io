HTML属性

我们之前在之前有一节html里面简单讨论过如何引入链接以及图片，那就算是html的属性，不过我们这里会更多的介绍一些，当然不会介绍太多，后续会出attribute_2...来补全更多的属性

我们在tags（标签）那个html中说过元素=标签+内容。但是看：<a href="https://chao0425.github.io/html&css&javascript/">对外展示的名字</a>，其除了“a标签”以及“对外展示的名字”内容外，还有一些其他的东西，即href。这个就是属性，为html的元素提供附加信息。

这个属性一般是加在标签的开始标签，就是说如果是成对的标签，就把属性加前面那个标签里面。

属性的格式一般为属性名="属性值"，当然，这个引号可以为单引号，不过具体什么时候必须用什么引号，以后再说。
html对于属性的名以及值大小写都是不敏感的，大小写都可以。不过我们还是强烈建议使用小写

光标悬停在我这里试试

好多属性都是涉及到css以及JavaScript什么的，比较麻烦，这里总结一下我学到现在常用的一些属性。
href链接(适用于a标签以及link标签)
src是源文件（适用于img标签等）、alt是图片无法展示时候展示的文字（适用于img标签）、width是宽度、height是高度。
title是光标悬停时候展示的文字（适用于所有标签）、style主要是css（适用于所有标签），不过里面可以设置个color，将文字变成color的颜色
