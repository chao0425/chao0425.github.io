我们对于在html中，对css的引用，可以分为3种：
  • 内联样式-在HTML元素中使用"style" 属性
  • 内部样式表 -在HTML文档头部head区域使用style元素 来包含CSS
  • 外部引用-使用外部CSS文件
    • 无论出于便捷、复用性、可维护哪个角度都是外部css文件更合适

    对于最简单的css就是，在标签里面给一个style属性就好了。
    这个作用域为整个整个标签，还可以设置举例边框多少个宽度（px为单位）如：margin-left:20px
