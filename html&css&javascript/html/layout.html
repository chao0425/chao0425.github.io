这部分则是内容了，对于id、style的解读，与上面的注释里面的解读是一样的效果。至于float则是挨着前面的float悬浮。
这个clear:both就是清楚前面的float效果。还有就是文字居中显示
我们发现div的块儿是可以嵌套块儿的。外层的块儿定义一个长宽出来（或者只定义长，亦或者只定义宽都可以）那么内部的块则以外部块儿的长宽为基准了
至于布局时候用的float属于css部分的内容，即浮动。可以使得块在一排展示（默认的换行的，float则不换行，相当于浮在上空）。具体效果可以尝试去掉，加回来等来对比看效果。
如果希望前面有float效果，后续没有，则可以用一个div里面的css写clear:both;就好了。
