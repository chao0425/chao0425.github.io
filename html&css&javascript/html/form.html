HTML 表单通常包含各种输入字段(输入框)、复选框(多选)、单选按钮(单选)、下拉列表等元素(下拉选择)。






我们首先使用form标签来创建一个表单，然后再在这个表单里面进行具体操作。(这个表单相当于调查问卷，只是一张纸，而问卷具体内容选项是我们在表单中写的)
接下来是对label的解释：for=""这个for内容与id是强相关的。假设id为xxx的输入框，我们for="xxx".单击这个label，则会直接可以开始输入了。
然后就是input：
input的type是指定输入类型，如果是text就是正常文本，如果是password则是密码形式。（默认不可见）。
id则是结合前面的for，以及css、js等合作的。
name则是提交的名字。以一个什么样的名字提交表单。这个名字也是用来指定在哪些范围内的单选按钮、多选按钮的
required则表明这个是一个必选项。
checked代表的是默认填写的内容
对于我们input的value选项，则是上交表单后，他们识别到的结果。
单选是radio、多选使用的是checkbox。
对于下拉选项我们使用的是select,具体选项内容我们使用option
对于具体的表单玩儿法，可以看看本文的源码
