可以看到文字是溢出来了。我们接下来通过css来解决溢出的问题。
首先这个css属性为overflow
默认情况是：overflow: visible;但是会出现溢出的情况
接下来看看hidden会如何

如果我们使用overflow: auto;
当可能出现溢出的时候，自动加上了滚动条。这样就不会造成溢出。

这个对于overflow: scroll;
那么溢出时候也是有滚动条
不过这个需不需要滚动条的时候他都存在占用空间
综合来讲，auto更好用一些。
