讲解一些类选择器、尺寸什么的

h1, strong {
color: black;
}
这样的话就是对于strong、h1标签都设置一个颜色就是black，而不用分开写，不大会影响可读性，而且还减少了代码量（不过面临的问题就是，如果分开写，那么h1的所有属性都在里面，而不用去找多个）

对于css的写法可以有如下几种例子(没涉及id)
p{ }: 为所有 p 元素指定一个样式。这个是对所有p都生效的
.marked{ }: 为所有 class="marked" 的元素指定一个样式。这个是对指定了class为marked生效的
.marked p{ }: 为所有 class="marked" 元素内的 p 元素指定一个样式。这个可能是div是marked的，而只对指定这个类的div内部的p生效，对于其他的不生效
p.marked{ }: 为所有 class="marked" 的 p 元素指定一个样式。这个只对p指定了这个类生效的css

尺寸我没什么想说的，我个人习惯使用px罢了。简单易懂。
