对于background我们可以设置color、image（可以配合url）、size（主要是image的size，如cover就很不错）attachment（设置为fixed保证画面不动，就很美妙）等

我们在设计background-img时候，要注意他的艺术，不要影响了我们本身内容的表达（我这个做的就不是很好）
我这个就是引入一个 background-image: url(../../image/css背景.jpg);作为背景。
然后background-size: cover;实现了背景协调的占满界面
background-attachment: fixed;保证这个图片协调的固定住不变
font-weight是可以调节字体的宽度的（是否加黑加粗）

background-repeat是说背景图片是否重复
background-attachment是说图片是否固定不动
background-position则是选择背景在哪里放着

最后补充一嘴，background可以缩成一个语句：“background: #fffaf0 url(../../image/css背景.jpg) no-repeat fixed 0%/cover ;”
具体就看background.css我怎么写的以及注释内容吧
