盒子模型

Margin(外边距):清除边框外的区域，外边距是透明的。与其他区域分隔的宽度。
Border(边框):围绕在内边距和内容外的边框。整个边框。
Padding(内边距):清除内容周围的区域，内边距是透明的。这个是用来占空间的。
Content(内容):盒子的内容，显示文本和图像。即具体的内容。

刚刚简单的展示了一下margin为0px的情况，可以看到两个颜色的背景是无色衔接的。（div的margin好像默认是0，不过p好像是有margin的）

我们设置width以及height时候，只是设置的内容的高度以及宽度，至于margin，padding以及border的宽度都是要自己设置的。
