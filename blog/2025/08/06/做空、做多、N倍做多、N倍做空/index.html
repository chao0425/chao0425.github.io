做空、做多、N倍做多、N倍做空

简单来说说股票中的做空、做多、N倍做空、N倍做多什么意思

先来说说做多

做多

这个比较简单，就是我认为某个基金、股票、指数未来会涨起来，我直接就买入他就好了。未来涨了再卖掉。

这个风险可以说是所有操作中风险比较小的了。毕竟只要买入的股票不推市，也不至于亏到分为不剩。

如：我在某只股票10元1股时候买入1000股，投入了1w元。
盈利：某一天，这只股票20元1股，全部卖出，应该是2w元，扣除手续费什么的杂七杂八暂且不计，盈利了1w元。
亏损：某一天，这只股票5元1股了，全部卖出，应该是5000元，同样的手续费暂且不计入，那么亏损了5000元。

做空

这个概念类似于借来一个商品，最后再把这个商品还回去，赚取差价。即：高价时候借来一些，未来跌了再还回去。

我们之前说做多的风险比较少是因为大概率不会亏损到血本无归，可是做空就不一定了。我们接下来还是简单的举例。

如：这只股票还是10元1股，我借入1000股，借了1w元。
盈利：某一天，这只股票跌到5元1股了，我全部归还，此时我只需要还5000元，我借了1w，归还5000元，不算手续费等，赚了5000。
亏损：某一天，这只股票涨到20元1股了，我全部归还，此时我需要还款2w元，借了1w，归还2w元，不算手续费等，亏损了1w元。如果我们做多的话，亏损1w元已经结束了，但是做空远不止于此，我们考虑：这只股票涨到30元1股了呢？显然，需要归还3万元，这样，亏损了2万元。

我们刚刚了解了基本的做多、做空，接下来聊聊N倍做多、N倍做空。为了数据的直观，我们以N == 3为例子。即3倍做空、3倍做多

N倍做多，N == 3

如果我们N倍做多的话，就相当于加入了杠杆（少量的资源撬动大量的资源，不过这个风险是高于普通的做空的，当然了，收益也是更大的）

这个3倍做空相当于你投入了1份钱，证券机构帮你再借2份钱

还是来举例子

如：我们某只股票10元1股，我投入了1w元3倍做多，这样的话，相当于机构再去借了2w元，合计我们买入了3000股，其中1000股是我们自己的1w买入的。
盈利：有一天，这只股票20元1股了，此时我们可以将这3000股全部卖出，即6w元，然后机构收回他帮你借来的2w元，我们自己的本金是1w元，不算手续费等费用，相当于我们净赚了3w元，同样的涨幅，再看我们做多时候举的例子，赚了1w元，此时我们赚了3w元，3倍做多，3倍收益。

看到这里，可能我们会想：这好啊，3倍收益。更快的盈利，虽说风浪越大鱼越贵，但是鱼贵的同时，我们不要忘了风浪很大啊。接下来聊聊“风浪”即风险

亏损：我们之前说，如果做多不加杠杆，只要买入的这只股票不退市，我们就不会血本无归。但是N倍做多则可能让我们血本无归（不过不会倒欠钱）
还是那只股票，10元1股票，投入1w元3倍做多，机构投入2w元。根据我们之前的例子，可能你会想，我接下来举的例子是股票跌到5元一股了，哈哈哈，不是的，你根本没有机会等到5元1股，这只股票剩下6.66元一股的时候，你已经血本无归出去了哈哈哈。我们来看看为什么6.66元就已经出局了。
考虑到我们是3倍做多，我们用1w块钱拿到了3000股10元一股的股票，如果此时股票跌落到了6.66元一股，此时3000 * 6.66 ≈ 2w。注意，证券机构帮我们借了2w元，机构是不允许自己亏损的。所以他会强制平仓，当你手中的股票价值刚好剩余他借来的钱的时候，他会把所有股票买掉，来避免自己的损失。
我们发现，从10元到6.66元是跌了33.33%就造成了我们亏损了100%，所以N倍做多赚N倍的时候，我们也是N倍亏损的。

N倍做空，N == 3

相信我们之前说完N倍做多后，我们已经意识到N倍的话，收益亏损都是N倍的。那么最后再来看N倍做空，同样以3倍为例。

依旧是我们之前的例子，股票10元1股，(借来)投入资金仍旧1w元。机构帮助你借了2w元
那么我们相当于是借来的1000股，机构帮你借来2000股，相当于3000股，3w元。

盈利：当某天股票价格跌落到了5元1股的时候，此时我们将3000股股票归还，即1.5w元，机构的操作是：得到3w元(1w本金，2w借来的)，归还的时候还回了1.5w元，那么赚了1.5w元，机构收取手续费后，将赚来的1.5w元还给你。即我们净赚了1.5w元，而上述普通的做多，我们赚了5000元，还是3倍收益。

亏损：当某天股票价格涨到20元1股的时候，我们归还3000股股票，这时候20*3000=6w，我们需要归还6w元，证券机构眼中的操作是：获得3w元(用户借来1w，帮助用户借了1w)，现在我们要还6w，把这借来的3w原封不动的还回去后，还要再还3w元，这3w元应该是用户自掏腰包还上的（机构不至于傻到自己垫钱，他只是赚个手续费、管理费等罢了）。所以这样来看，我们是亏损了3w元，对比上面说的普通做空，我们亏损了1w元，这样是亏损了3倍的价格。

上述概念后的随谈

N倍做多的话，为什么不选择自己投入N倍的资金，这样的话不会被强制平仓，更稳妥啊。

答：如果我本身只有1w元，但是我希望获得大额的收益，那么自然可以选择的是N倍做多，如果盈利的话，那就是1w元办了3w元的事儿。

做空的话，如果股票价格飞涨，还不起了怎么办？

答：借股票前，机构会评判你的资质，类似于银行是否会将贷款借给你。或者说，机构会要求你交一笔保证金，在上述的例子中，10元1股的股票借入1000股，即1w元，机构可能会要求你缴1.5w的保证金，当股票价格15元1股的时候，他会强行的用你这1.5w来还回股票，来保证自己不受亏损。

那么我们综合来看：无论是做空、N倍做多、N倍做空，机构都是会有着一个强制平仓止损的措施的。

趣味思考：我们之前说了N倍做多，前两天看到一个评论说“有没有20倍做多xxx”，那么他的风险会是怎样的呢？他能接受的市场回撤大概是多少？

所以市场上常见的一般是2、3倍的做空做多，毕竟市场大多数大的回撤一般也就30%，由于市场是不确定的，过去的历史不能代表未来，所以出现黑天鹅事件也是情理之中，3倍做多、2倍做多也是可能会亏得血本无归的，只不过概率是小于20倍做多。N倍做多的情况，可以接受的回撤是1/N * 100%
